# Knowledge Wiki

This directory holds compiled knowledge pages.

These pages are maintained synthesis artifacts, not raw sources and not live
control-plane authority.

## Typical Page Kinds

- concept page
- entity page
- boundary note
- comparison page
- topology map
- timeline summary

## Rules

- compiled pages should link back to the source packets they synthesize
- compiled pages may be rewritten as understanding improves
- compiled pages may note contradiction, uncertainty, or stale claims
- compiled pages should not present themselves as live tracker or lane truth

- `tusk-control-plane.md`

- `upstream-boundary-map.md`
- `automation-timeline.md`

## Query Rule

If a good answer emerges from the wiki and should persist, file it back here as
an update or a new page instead of leaving it only in chat history.
